    选择提供商

    配置参数

    从 OpenAI 平台获取 API Key

    默认地址：https://api.openai.com/v1

    从下拉列表选择，或直接输入自定义模型名称（如：qwen-plus、claude-3-sonnet 等）

    精确（0） 平衡（0.5） 创造（1）

    配置帮助

    • •OpenAI：使用 GPT-4 系列模型，需要有效的 API Key
    • •DeepSeek：国产大模型，性价比高，支持中文
    • •本地模型：使用 Ollama 等本地部署的模型，地址填 http://localhost:11434
    • •Azure OpenAI：企业用户使用微软 Azure 托管的 OpenAI
    • •自定义：支持任何兼容 OpenAI API 格式的模型服务
